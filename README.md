# Portfolio Website

(Outdated as an advanced version is currently in development)

A full-stack portfolio website showcasing projects, experience, and technical skills. Includes a custom library system, secure admin login, and real-time weather API integration.

---

## Features
- Responsive front-end with HTML, CSS, and JavaScript  
- Full-stack functionality with PHP and MySQL  
- Secure admin login using PHP sessions and SQL validation  
- Custom library system for managing content  
- Real-time weather API integration  
- Clean, user-friendly interface  

---

## Tech Stack
- **HTML, CSS, JavaScript** (Front-End)  
- **PHP** (Back-End)  
- **MySQL** (Database)  
- **VS Code**, **Git**  

---

## Testing
- Admin authentication and session management checks  
- SQL validation and injection prevention  
- API data fetch verification  
- Input validation and UI responsiveness testing  

---

## Installation
- git clone this repository  
- Import the MySQL database into your local server  
- Configure database credentials in `/php/config.php`  
- Open in a local or hosted server to run the website
